# 🚀 Modern Full Stack Developer Portfolio

A stunning, high-performance portfolio website built with cutting-edge technologies and smooth animations.

## ✨ Features

- **Smooth Animations**: Framer Motion for butter-smooth interactions
- **Modern Aesthetic**: Dark theme with purple glows and glassmorphism
- **Fully Responsive**: Mobile-first design that works on all devices
- **Performance Optimized**: Fast loading, optimized images, code splitting
- **SEO Ready**: Meta tags, semantic HTML, structured data
- **TypeScript**: Full type safety across the application
- **Tailwind CSS**: Utility-first styling for rapid development

## 🎨 Sections

- **Hero**: Eye-catching introduction with animated gradient backgrounds
- **Projects**: Showcase of featured work with smooth hover effects
- **Skills**: Interactive skill bars and experience stats
- **Contact**: Beautiful contact form with social links

## 🛠️ Tech Stack

- **Frontend**: React 18, TypeScript
- **Build**: Vite
- **Styling**: Tailwind CSS, Framer Motion
- **Icons**: Lucide React
- **Animations**: Framer Motion

## 📦 Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🚀 Getting Started

1. Clone or download this project
2. Run `npm install` to install all dependencies
3. Run `npm run dev` to start the development server
4. Open `http://localhost:5173` in your browser

## 📝 Customization

- Update personal information in component files
- Modify project data in `src/components/Projects.tsx`
- Adjust color scheme in `tailwind.config.js`
- Add your social links in `src/components/Contact.tsx`
